# smsweb
*Text messaging tool similar to WhatsApp web using your smartphone as a server.*
---
Webserver based on Nanohttpd: https://github.com/NanoHttpd/nanohttpd
>#### Features
- Conversations
- Bulk SMS
- Group SMS
- Clipboard Sync
- Realtime Sync
- JSON Based API

>#### Goals
- Addation of security in a multi-user environment.
- A better interface.
